To run:
```shell
$ mdkir communicator && cd communicator
$ git clone https://github.com/NorteX-dev/communicator.git
$ yarn
$ yarn start
# or
$ npm install
$ npm start
```

To push changes:
```shell
$ git init -b master # unless already done
$ git add .
$ git remote add origin https://github.com/NorteX-dev/communicator.git
$ git remote -v # Verify
$ git push origin master
```
